🚀 БЫСТРЫЙ СТАРТ:

  1. Загрузить папку POPO на GitHub:
     cd POPO
     git init
     git add .
     git commit -m "Initial commit"
     git remote add origin YOUR_REPO_URL
     git push -u origin main

  2. Зайти на vercel.com/new

  3. Import Git Repository

  4. Добавить Environment Variables:
     - DATABASE_URL
     - TELEGRAM_BOT_TOKEN
     - TELEGRAM_CHAT_ID

  5. Нажать Deploy!
